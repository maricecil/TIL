# 4. 전역변수
## 프로그램 전체에서 접근할 수 있는 변수로 함수 외부에서 선언됨
```python
x = 10  # 전역 변수
def my_function():
    print(x)  # 함수 내부에서도 전역 변수 x에 접근 가능
my_function()  # 출력: 10
print(x)  # 출력: 10
```
